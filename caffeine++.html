Caffeine App has been created to help you track caffeine consumption and requires its users to allow Health Access for storing data and displaying daily cups total. Caffeine++ doesn’t require any personal user details to be entered anywhere in the app. Caffeine++ App and it’s creators can not be held responsible for any incorrect user data entered in the app. Any user generated data that is entered in Caffeine++ App is neither saved on a server nor distributed. User generated data is exclusively stored on a user’s iPhone devices, though it will be included in an iCloud backup or local iTunes backup when encryption is enabled.
